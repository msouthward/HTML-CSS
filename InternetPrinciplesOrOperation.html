Internet Principles Or Operation

Home

Modem - Modulator-demodulator - Turns inbound connection into an ethernet connection
Browser - Program you use to access the World Wide Web
Protocol - An agreed format or set of rules to transmit and receive data
Internet - A vast network that connects computers all over the world
World Wide Web - A large group of interconnected public web pages or information that is accessible through the internet
Internet service provider - A company that provides access to the internet for a fee
Web server - A web server is a software and hardware that uses HTTP and other protocols to respond to client requests made over the world wide web
Proxy server - An intermediary server separating the user and the destinations that they are browsing.
Packet - A unit of data that can be sent across a network. A typical packet contains perhaps 1,000 or 1,500 bytes.
Packet loss - Caused by interference, usually across wireless networks, or network congestion.
URL - Uniform resource locator. The address of a resource on the web
TCP - Transmission Control Protocol - Standard that enables programs to send packets across the internet and ensure the successful delivery of data
IP Address - USed for sending data from one device to another across the internet. Every device has an IP address that uniquely identifies it and enables it to communicate with and exchange data with other devices connected to the internet
MAC Address - Media access control address is a unique identifier for each device and never changes.
Cookies - Cookies are tiny pieces of data downloaded by your computer when you visit a website. Some cookies track where you have been online. Some cookies store sensitive data such as credit card numbers.
DNS - Domain name server - Web server’s IP addresses are associated with their domain name. When entering on a domain name, the request will go to a DNS server usually associated with the ISP. The DNS server will check if it has the address saved. If not, it will ask other DNS server’s and then return the address.
