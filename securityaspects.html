Malware - Malicious software
Virus - A type of computer program that, needs user input to replicate itself. It replicates itself modifying other computer programs and inserting its own code.
Ransomware - Takes your files and blackmails you
Spyware - Steals your data
Adware - Spams you with ads
Worms - Spread across computers
Trojans - Sneak malware onto your PC
Botnets - Turns your computer into a zombie
White hat = Good
Black hat = Bad
DoS or DDoS is flooding a server so it can’t be used
Antivirus software - Software that is created specifically to help detect, prevent and remove malware
Firewall - Firewalls can be software, hardware or both. There are packet filters, stateful inspection and proxy server firewalls. Data is sent to the proxy so it will filter it and if it is unsafe it won’t send it to you.
